# SHL AI Agent

An AI-powered assessment recommendation system built using FastAPI and Sentence Transformers.

## Features
- Semantic search for SHL assessments
- FastAPI backend
- AI-based recommendations
- Clarification handling for vague queries

## Tech Stack
- Python
- FastAPI
- Sentence Transformers
- FAISS
- JSON dataset

## Run Locally

```bash
pip install -r requirements.txt
cd app
uvicorn main:app --reload
